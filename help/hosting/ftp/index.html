FTP and SFTP

Help » Hosting » FTP and SFTP

Siteleaf allows you to publish to any web host that supports standard FTP or SFTP (also known as Secure or SSH File Transfer Protocol).

These hosts include:

  • Media Temple
  • DreamHost
  • 1&1
  • Bluehost
  • GoDaddy
  • HostGator
  • and more

Settings

Benefits

Publishing to your own FTP or SFTP server gives you full control over your files, and (depending on your web host) the ability to use .htaccess files to set up 301 redirects or password protect your published content.

Limitations

Siteleaf currently supports the FTP and SFTP protocols only, not FTPS.

Siteleaf subdomains (e.g. yourname.siteleaf.net) are not supported with FTP or SFTP hosting.
